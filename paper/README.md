# Paper Materials

This folder contains the current paper draft and supporting planning files.

## Main Draft

- `manuscript.md`: complete Markdown paper draft with abstract, origin story, definitions, methods, experiments, results, limitations, conclusion, and references.
- `references.bib`: BibTeX references for the draft.

## Supporting Notes

- `outline.md`: paper structure and claim boundary.
- `methods_notes.md`: implementation-to-methods bridge.
- `figures_plan.md`: figure list, intended use, and caption drafts.

## Current Paper Position

The draft presents RDT Adaptive Hierarchies as a bounded methods paper, not a universal RDT theory. It cites the Zenodo RDT integer-depth preprint as the origin of the Recursive Division Tree line of work, then explains how this repository narrows the idea into a benchmarked hierarchy framework.

The draft is suitable for technical review. Before submission, it still needs stronger external baselines, confidence intervals, and a cleaned publication format.
