# Changelog

## 0.1.0 - 2026-05-18

- Created the initial public research package for RDT Adaptive Hierarchies.
- Added the `src/rdt_adaptive_hierarchy` package with core hierarchy, stable partitioning, coverage, residual sampling, geometry validation, and baseline modules.
- Added runnable examples, pytest coverage, benchmark entry points, concise result summaries, references, and reproducibility notes.
- Added expanded validation for stable partitioning, RDT-cover, residual sampling, geometry validation, and performance scaling.
- Added Halton and Hypothesis coverage support, expanded RDT-cover predicates, stable-partition null controls, and geometry integral checks.
- Added public links to the companion RDT Spatial Index repository for spatial-indexing provenance.
- Bounded the public claims to stable partitioning, deterministic coverage, known-form geometry validation, and research-only residual sampling.
