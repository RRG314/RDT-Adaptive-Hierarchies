# Results

This directory separates concise interpretation from raw artifacts.

`summary_tables/` contains short result summaries intended for human review.

`raw/` contains JSON, CSV, and Markdown outputs from the formalization run. These files are kept so benchmark claims can be traced to concrete artifacts.

The summary tables do not replace rerunning the benchmarks. They record the current evidence snapshot from 2026-05-18.
